Why Update Software? *shakes Head*

~E

Why?  Why update Adobe Reader when they release a new version?  Oh, there’s a new version of WinAMP.  Hey look, my Quicktime is out of date.

Don’t update.  Almost all these products, when updated, become horrendously large filesizes and product size.  Not to mention how they all are trying to sneak in some type of Toolbar or other applications for “Review”.  The most annoying always is the RealOne player. 

Not to mention how these applications always think they are the best for playing ALL of your media, movie files, MP3s, etc so they always assume we want Real Player to open my mp3 files and play them.  NO!!!

Not to mention all of that time you fought to remove them from the StartUp procedure (running in the tooltray, in the background for NO reason) is gone because when update them, they feel the need to load themselves in the background for “quicker launching”.  I’m sorry, I hardly every watch MOV files online.  So why would I want QuickTime running in my tooltray, along with 15 other players and applications?

Now I’m talking updating software like media players, readers, Flash/multimedia, etc.  Those little applications we need to browse the web daily and listen to audio files and such.  Obviously mainstream applications need updates to fix bugs and large new features.

I have yet to see a reason for me to upgrade to the latest Adobe.  I’m running Adobe Reader 5.0.  Small, lightweight, loads fairly quickly.  I really preferred version 4.0 of the reader as it was fast.  But I have Adobe’s PDF creator software, which is version 5.0.  So I’m on 5.0 now.

I’m running WinAMP 2.61.  No, that’s not a misprint.  Yes, I’ve been running the same verion since 2000.  Why?  Why not?  All shoutcast streams work just fine, as well as MP3 playing.  I don’t want WinAMP to take over as my Media Player of choice, so there’s no reason to upgrade.  Heck, even plug-ins for WinAMP 5.0 work just fine in 2.61!  Also, 2.61 doesn’t force “reporting” or behind-the-scenes of contacting WinAMP to report back on statistics.  It’s extremely lightweight, and very non-intrusive.

AOL Instant Messenger.  I’m not sure what the version number is, but I’ve been using the same messenger since 2000!  Why?  It’s fast, lightweight, and doesn’t have those annoying talking ads!  Actually I no longer use AIM’s client (nor ICQ or Yahoo) as running all of them was too bulky.  I now use Trillian.  And the only reason I keep updating Trillian is because one of the IM servers will change something, so I have to update to get a version that works.

Now RealOne needs to be slapped.  I tried to run 8.0 of their player because they changed the name to RealOne.  Then a lot of content because RealOne-only content so I had to update.  Get this, Real has figured out a way to force users to upgrade.  Their players expire!  Even though I haven’t seen one expire in the past 6 months (maybe the consumers got pissed off, like me).  Perhaps that force upgrade was because it was the first RealOne release.

Microsoft products
Ok, now these I do update as often as I can as they just get better and betters.  Like Office and WMP and MSN Money, MSN Messenger, etc.  I don’t like how bloated Windows Media Player 10 has gotten, but they always seem to be improving performance, along with adding features (unlike ANY of the others above mentioned).  With a little tweaking of the settings, WMP10 can load right up to display the library or Playing Now tab instead of the default Guide or What’s Hot webpages (where it contacts MS and downloads content).  Then it loads quickly.

I’d actually like to see Microsoft release a lightweight Media Player, like the old 6.4 version (that works with almost ALL content right now, btw!).  Maybe call it Windows Media Player Light” that is strictly used for web browsing and streaming.  Loading WMP10, the library of MP3 files I have, checking for updates, etc seems quite large for an application that I simply want to stream a Shoutcast stream with and run in the background while I work (and taking up almost 25MB of memory!).

WMP10 is still too bulky for me to load in the background while I work.  So I still use WinAMP 2.61 in the background to stream my MP3 collection and stream the ShoutCast stations I’ve come to love.

 

The moral of the story is if you see a popup that says, “Your X program is out of date” or “There is a newer version of X program”, don’t upgrade.  Click the “Don’t remind me” button and keep working.
